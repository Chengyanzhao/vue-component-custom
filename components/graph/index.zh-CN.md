
## API

### triangle

| 成员      | 说明                                        | 类型                              | 默认值 |
| --------- | ------------------------------------------- | --------------------------------- | ------ |
| direction | 顶角方向，可选`top` `right` `bottom` `left` | string                            | top    |
| bgColor   | 图形背景颜色                                | string                            | -      |
| size      | 顶角点至四边的距离                          | number \| {top,right,bottom,left} | 50     |

### 事件

| 事件名称 | 说明             | 回调参数        |
| -------- | ---------------- | --------------- |
| click    | 点击图形时的回调 | (event) => void |
